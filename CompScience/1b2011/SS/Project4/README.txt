Author: Jeroen Hofman
Date: December 18th, 2011
Content: Stochastic Simulation, Project 4, Travelling Salesman Problem.

This zip file contains the following:
- TSP_gen.cpp/TSP_sim.cpp, the core code for the genetic algorithm, simulated annealing respectively, compile with make gen/sim respectively.
- Data, some coordinate maps from TSPLIB, output from the above files also gets stored there and can be processed by TSP.nb.
- Report, the file containing the report, both .tex and .pdf.
- SFMT, Mersenne Twister libraries.
- GA_Pointparticles.pdf, a paper on using genetic algorithms to solve the charges on a sphere problem.
